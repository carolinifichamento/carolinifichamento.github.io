texto nutrição

A Nutrição é o ato de fornecer componentes fundamentais

para organismos e células em pleno funcionamento.

Logo, a Nutrição é uma área que domina os nutrientes

que os seres precisam consumir

além de otimizar o consumo com as melhores combinações

para o máximo aproveitamento possível

destes alimentos
